# Mini Geometry Calculator 

## Project Description

This is a simple JavaScript tool that calculates the area and perimeter of a rectangle.  
It is designed for quick measurements, such as estimating room size for flooring or skirting.

## How to Run
1. Open `index.html` in a web browser.
2. Enter the length and width when prompted.
3. Open the browser console to view the results.

## Formulas Used
- Area = length × width
- Perimeter = 2 × (length + width)
